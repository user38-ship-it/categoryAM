# GitHub Pages

Ця папка підготовлена для публікації як статичний сайт.

Що потрібно для сайту:
- `index.html`
- `category_tree_viewer.html`

Як увімкнути Pages:
1. Відкрити `Settings` -> `Pages`
2. Обрати `Deploy from a branch`
3. Вибрати branch `main`
4. Вибрати folder `/ (root)`
5. Зберегти зміни

Примітка:
- `category_tree_viewer.html` є основним застосунком.
- Якщо його ще не завантажено в репозиторій, сайт не запуститься повністю.
